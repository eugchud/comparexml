CompareXML
=============================

Программное решение «Сравнение XML-файлов» предназначено для поиска и подсчета различий в структуре XML-документов.
Решение производит сравнение двух XML-файлов, по результатам которого для каждого из элементов в каждом из файлов производится определение наличия либо отсутствия аналогичного элемента в дргом файле. Также в случае наличия различий среди потомков элемента производится их подсчет. Вывод результатов сравнения производится при помощи цветовой схемы.

Скомпилированная версия программного решения доступна по ссылке: https://drive.google.com/open?id=1OpJdiuB2j6TI0dHAtbsZ0lFah43-DWT6
